Cours suivi

Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Définition: la toile, est l'ensemble de liens hypetexts.

Internet

Définition: Le réseau qui permet la communication entre les ordinateurs, dans ce réseau se trouve plusieurs services.

Cloud

Définition: le stockage d'internet, il est là pour stocker vos données et services.

W3C

Définition: le W3C est l'ensemble des règles du web, des protocoles ajoutés.

Partie 2 - Distinguez les langages du Web

Langages client

Définition: HTML, CSS, JS, ce sont les langages qui vont structurer le site que nous allons parcourir.

Navigateurs

Définition: outils pour traduire le langages client sous forme de sites web.

Langages serveur

Définition: comme le nom l'indique le client n'y touche pas, ils indiquent au serveur comment le site doit se comporter.

Frameworks

Définition: boites à outils maintenant indispensable pour la création de sites web, il y en a un exprès pour chaque langages serveur.

Bases de données

Définition: stockage de donnée d'un site web.

Site responsive

Définition: site web dont le contenu s'adapte à la largeur d'écran.

Application mobile

Définition: c'est faire un site dédié pour le mobile

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Définition: les serveurs sont les machines qui possèdent et qui distribuent aux clients les sites web.

Adresse IP

Définition: suite de nombre qui est propre a chaque ordinateur.

Nom d’hôte et DNS

Définition: traduction de l'adresse ip.

Protocoles

Définition: langage de communication entre machines.

Partie 4 - Identifiez les métiers du web

Développeurs

Définition: le monsieur qui conçoit, teste, documente et maintient le produit.

Product Managers et chefs de projets

Définition: le monsieur qui met en place une statégie par rapport aux besoins du marché.

Designers

Définition: le monsieur qui gère l'habillage du site web.

Testeurs

Définition: le monsieur qui test :/

Techniciens informatique

Définition: le monsieur qui fait en sorte que tous travailles dans de bonne condition, gère la maintenance du matériel.

Boîte à outils du développeur

  • IDE comme VS Code : L'IDE est un éditeur de text regroupant des outils pour les devs.
  • Github : applications qui peut servir de cloud pour projets, endroits où l'on peut publié nos projet et aussi heberger un petit sites web.
  • Librairies / bibliothèques : les Librairies/bibliothèques sont des fonctions faites que n'importe qui peut utilisé.
  • Frameworks : boites à outils pour les devs Back-End, il en existe plusieurs chacun propre à un langage en particulier comme, php symphony, c# ASP NET et java angular.
